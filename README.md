# Creating new module

To create new module, simply call `sh ./scripts/create-module.sh` in the root directory. You will be prompted for the name of new module. You can also put the name as an argument for the script.

# Renaming the solution

To rename the solution, simply call `sh ./scripts/rename-solution.sh` in the root directory. You will be prompted for the new name. You can also put the name as an argument for the script. It is advised to do it in early stage of development.
